# Bandit level 32 -> 33

## Objective
to find the password by escaping from the uppercase shell
## Commands Used
$0,cat /etc/bandit_pass/bandit33
## Explanation
when we enter into this level you will be in the upper case shell to escape from this shell we need to use  $0 command to get the normal bash shell there we can use cat /etc/bandit_pass/bandit33 this command to see the password
## Outcomes
learned how to escape from the shell
