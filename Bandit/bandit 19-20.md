# Bandit level 19 -> 20

## Objective
to find the password we need to run it as bandit20
## Commands Used
ls -l,./bandit20-do cat /etc/bandit_pass/bandit20
## Explanation
we need to run ls -l to see the permissions for that file you will see S in the permission which is setuserid which helps us to 
temporarily run the file use this command to run the file and to see password for next level ./bandit20-do cat /etc/bandit_pass/bandit20
## Outcomes 
learned about the setuid concept
