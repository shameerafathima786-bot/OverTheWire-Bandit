# Bandit level 0 ->1

## Objective
To retrive the password for the next level in the file
## Commands used
ls -- to list the files in the directory
cat -- to read the file
du -- to see the disk usage
file -- to see the type of file
cd -- to move to next directory
## Explanation
In this level we need to use ls command to list the file in home directory and by using cat command we can read the file and password for the next level
